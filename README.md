# CSC350 Final Project
## QRS Quality Rental Service

---

### Minimum Viable Project
- Have users be able to create an account or sign in with an existing account
- Users with accounts are able to rent a car on specific dates

---

### Features
- Sign up and login functions
- A mobile first responsive design
- Search and filtering for car inventory
- Users are able to manage or update bookings

---

### Tech Stack

- Frontend: HTML, CSS, PHP
- Backend: PHP
- Database: MySQL
- Version control through Github

---

### Our Process

- The process for creating our website was each contributor developed seperate features in their own local branch. When a feature was finished the branch would be pushed to github where a pull request was created and another developer had to review and approve changes before the feature was implemented to the main branch.
